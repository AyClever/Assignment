   -The git init: The git init command is use to create a new git repository folder or use an existing one e.g

	Initialized empty Git repository in C:/Users/AYO/Desktop/Assignment/.git/
   -The Git Add .: The git add is a command, which adds changes in the working directory to the staging area. 
    With the help of this command, you tell Git that you want to add updates to a certain file in the next commit.e.g
	C:\Users\AYO\Desktop\Assignment>git add .

	C:\Users\AYO\Desktop\Assignment>
   -The git commit -am "Initial Explain": The git commit -am command is used to tell Git to save your changes to the local repository.e.g
	C:\Users\AYO\Desktop\Assignment>git commit -am "Initial Explain"
